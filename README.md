# EcoTask 🌱

Proyecto desarrollado para la materia Programación Web.

## Objetivo

EcoTask permite registrar acciones ecológicas realizadas por los usuarios para fomentar el cuidado del medio ambiente.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- JSP
- Java Servlets
- MySQL
- Apache Tomcat
- Maven

## Funcionalidades

- Registro de usuarios
- Inicio de sesión
- Dashboard
- Registro de acciones ecológicas
- Base de datos MySQL
- Repositorio GitHub
